## 3. Components Library

### 3.1 UI Components (shadcn/ui)
- Button, Input, Select, Textarea
- Card, Dialog, Drawer, Sheet
- Table, DataTable
- Tabs, Accordion
- Avatar, Badge, Alert
- Toast, Tooltip
- Calendar, DatePicker
- DropdownMenu, Command

### 3.2 Custom Components

| Component | Description |
|-----------|-------------|
| `PropertyCard` | Card listing properti dengan gambar, harga, lokasi |
| `RoomCard` | Card kamar dengan status dan harga |
| `ResidentCard` | Card penghuni dengan info singkat |
| `PaymentTable` | Tabel riwayat pembayaran dengan filter |
| `ChatBubble` | Bubble message (text/image) |
| `ImageUploader` | Upload gambar dengan preview |
| `MapPicker` | Google Maps location picker |
| `StatCard` | Card statistik dengan ikon dan value |
| `FilterBar` | Bar filter untuk marketplace |
| `SearchBar` | Search dengan autocomplete |

## 4. Form Specifications

### 4.1 Registration Form (Owner)
```typescript
interface OwnerRegistrationForm {
  // Required
  name: string;           // min 3 chars
  email: string;          // valid email
  phone: string;          // +62 format
  password: string;       // min 8, 1 uppercase, 1 number
  confirmPassword: string;
  ktpImage: File;         // max 5MB, JPG/PNG
  selfieImage: File;      // max 5MB, JPG/PNG
  businessName: string;   // min 3 chars
  businessAddress: string;// min 10 chars
  propertyType: 'KOS' | 'KONTRAKAN' | 'VILLA' | 'HOUSE' | 'LAND';
  
  // Optional
  npwp?: string;
}
```

### 4.2 Property Form
```typescript
interface PropertyForm {
  name: string;
  type: PropertyType;
  description: string;
  address: string;
  province: string;
  city: string;
  district?: string;
  postalCode?: string;
  latitude: number;
  longitude: number;
  landArea?: number;
  buildingArea?: number;
  totalFloors?: number;
  yearBuilt?: number;
  certificate?: CertificateType;
  images: File[];          // min 5, max 20
}
```

### 4.3 Resident Form
```typescript
interface ResidentForm {
  // Personal (Required)
  fullName: string;
  ktpNumber: string;
  ktpImage: File;
  selfieImage?: File;
  dateOfBirth: Date;
  placeOfBirth: string;
  gender: 'MALE' | 'FEMALE';
  phone: string;
  email?: string;
  occupation?: string;
  originAddress: string;
  
  // Emergency Contact
  emergencyName: string;
  emergencyRelation: string;
  emergencyPhone: string;
  
  // Rental
  roomId: string;
  checkInDate: Date;
  rentalPrice: number;
  depositAmount: number;
  paymentDueDay: number;    // 1-31
}
```

## 5. State Management

### 5.1 Zustand Stores
```typescript
// Auth Store
interface AuthStore {
  user: User | null;
  isLoading: boolean;
  login: (credentials) => Promise<void>;
  logout: () => Promise<void>;
  updateProfile: (data) => Promise<void>;
}

// Property Store
interface PropertyStore {
  properties: Property[];
  selectedProperty: Property | null;
  isLoading: boolean;
  fetchProperties: () => Promise<void>;
  selectProperty: (id: string) => void;
}

// Chat Store
interface ChatStore {
  chats: Chat[];
  activeChat: Chat | null;
  messages: Message[];
  sendMessage: (content: string) => Promise<void>;
  uploadImage: (file: File) => Promise<void>;
}
```
